Steps to Get Started

  1. Download the agent for your OS from above.
  2. For Ubuntu, after downloading, run the following commands in the terminal: 
    chmod +x ubuntu_agent
./ubuntu_agent
  3. The agent will start sending heartbeats to the server. View its status on the Server Dashboard.
